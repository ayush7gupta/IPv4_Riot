\subsection{Detailed Description}
Test for low-\/level P\+WM drivers. 

This test initializes the given P\+WM device to run at 1\+K\+Hz with a 1000 step resolution.

The P\+WM is then continuously oscillating it\textquotesingle{}s duty cycle between 0\% to 100\% every 1s on every channel.
